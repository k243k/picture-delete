SUPER EASY INSTALLATION - Choose ONE:
========================================

METHOD 1: Using setup.bat (Recommended)
----------------------------------------
1. Download ZIP from GitHub
2. Extract ZIP
3. Double-click "setup.bat"
4. Press Y when asked
5. Done!

METHOD 2: Using PowerShell
---------------------------
1. Download ZIP from GitHub
2. Extract ZIP
3. Right-click "INSTALL.ps1"
4. Select "Run with PowerShell"
5. Press Enter when asked
6. Done!
